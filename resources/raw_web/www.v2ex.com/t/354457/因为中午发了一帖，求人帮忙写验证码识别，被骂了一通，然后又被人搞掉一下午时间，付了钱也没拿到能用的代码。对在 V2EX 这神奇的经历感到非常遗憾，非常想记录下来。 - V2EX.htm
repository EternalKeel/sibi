V2EX  ›  水深火热

因为中午发了一帖，求人帮忙写验证码识别，被骂了一通，然后又被人搞掉一下午时间，付了钱也没拿到能用的代码。对在 V2EX 这神奇的经历感到非常遗憾，非常想记录下来。

  •   
  •   z903221074 · 111 天前 · 3448 次点击  
    这是一个创建于 111 天前的主题，其中的信息可能已经有所发展或是发生改变。

    首先，骂我也好，我来描述一下从中午开始发生的事情。

    学生，最近几天在准备即将到来的很重要考试，昨晚却收到任务要写一个网站的爬虫。嗯，忽略任务。

    今天上午被打电话催了一遍，看来也挺要紧的，打开目标网站看了下，登陆需要验证码，内部拿数据不难。

    我写代码的风格，就是对于自己不会的，赶紧上 Google 或者 Github 找代码搞过来用，花了一点时间，没搞定，心里却想着考试，真的挺重要的，就不写了。

    正做着模拟题，灵机一动想到要不直接去买代码，即省时间，又能直接对应着不修改就用，非常合适。

    赶紧跑到 V2EX 上发一帖， https://www.v2ex.com/t/354318

    哦哦，对了，顺便学了个 Shell 新技能给大家分享下：

    for k in $( seq 1 1000 )
	echo ${k}

    这样就可以在 Shell 中执行循环了，可以用这个结合 wget -O 参数来瞬间下载很多验证码，嗯，是不是很好用？

    好了，赶紧回到主题，我发那个帖子，考虑了下自己经济能力，又考虑下自己如果花时间用心写，一步一步跟着网上已有的指导做，还是能做出来的，按道理不会花很久。就自以为是的写上了（ 100-150 ）的报酬，然而没想到，这个价格会成为被鄙视的根源。

    的确，我给的这个价格，对此论坛在坐的程序员来说，太少了。

    真心认真求帮助，换来很多带歧视意味的回复。心里也有点失落和难受，根源应该是那个价格吧，我认为。

    正如 @zzcchh 说的，最好就不要提钱，也许大家会集思广益来帮这个忙，但是正因为提了钱，让很多看帖的人感受到了受鄙视的感觉。但其实，我作为一个学生，成为像论坛里很多人一样厉害的程序员，也是我的梦想，没什么本事的我怎敢鄙视素不相识的人。

    被很多人那样回复了以后，我打算放弃这个计划，还是靠自己写吧，感谢了一下帖子内提供帮助信息的朋友。

    却发现，，， V2EX 并不能删帖。。。

    好吧，赶紧补个 append ，想让这一切回归平静。

    当然，与上述的故事发生的同时，另一条故事线也在展开着

    在我写 append 的同时，一个微信好友请求让我非常激动，随加之。

    一看，靠谱啊， 99%成功率，用上深度学习出来的模块，完美的满足了那个工程的需求（不引入大型外部库，尽量使用 PIL ）。

    行，本来我在帖子里被骂了以后自己也觉得 100 多元太少，还是愿意给加钱的（如果有非常好的东西，何乐而不为？），便满怀激动的有了如上的对话。

    附上知乎的展示内容 http://www.zhihu.com/question/22479139/answer/120141781 哇，效果真的不错，很显然，我对于这个作品感到眼前一亮，非常想要使用，觉得值得为人家的工作多付一些钱。当然，买他的深度学习训练源码那肯定不够。但确实不能付出时间。

    经历过讨价还价， 300 成交了，他去忙着标记了，我 也开始 TPO 阅读了。

    半小时后， tpo 做到第二篇，为了适应考试，不应该中途退出。但是人家做完了，叫我了。

    怀着代码采用 Python 实现，对常见验证码的识别率达到 99%以上，速度可以满足工业化需求，总体效果能与国内外各知名公司的开源 OCR 程序抗衡，预处理的目的是将 95%的识别率提高到 99%，也就是说不采用任何预处理直接运行我的代码可以达到 95%的识别率， 我的程序对常用验证码的识别率已经达到了 99%以上，满足绝大多数工业化要求，所以暂时并不打算再做提升 这样令人激动的产品描述，我激动的打开他发来的测试图，考试的计时器还在运转，我扫了几眼，感觉全对了，迅速付款，准备一会儿做完考试题就赶紧上工业级的代码，而没想到，这，却为自己埋下了坑。

    付款加上提现手续费，是我认为的正常的微信转账对别人劳动的尊重

    而其实，此图中验证码的准确率只有，，，可怜的 60%。。。和知乎上说的不是一回儿事儿。

    做完了第二篇阅读，本来是还有一篇的，但我相信论坛中的各位大神也都能理解学生时代对代码的激动，我心没法静下来，决定去看代码。

    于是乎，如下

    后来，发现是一些包的版本不对，实际结果没这么惨，达到 50%了

    参照我想要的代码，这显然是不满足要求的

    https://lujq.me/about/ https://github.com/golmic

    又有了如上神奇的对话，我也是很无语的。不过我感觉，作为一个在 V2EX 上没什么实力，并且刚在另一个帖子被骂了一遍的我，还会在这个帖子中被骂一遍的。钱已经要不回来了，人家什么方案都不接受，并且保持沉默了。东西啥也没有，时间还消耗掉了整个下午和晚上。这当然算得上一次奇妙的经历！人家是数据爬取工程师，我菜我小白，站在他那一边的人当然多了，所以我认为我发完这个帖子还是会被再骂一遍的，不过，无所谓了，记录一下今天的事儿，今天的感受。

    文笔很差，不善言表， MarkDown 用的还不好，多包涵

    祝大家即将到来的周末愉快
