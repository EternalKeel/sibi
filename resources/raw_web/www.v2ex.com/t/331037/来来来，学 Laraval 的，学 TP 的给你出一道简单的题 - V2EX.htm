    这是一个创建于 216 天前的主题，其中的信息可能已经有所发展或是发生改变。

    目前新闻模块、文章模块要实现缓存功能：

    1 、新闻首页缓存 100 秒

    2 、新闻内页缓存 1 天

    3 、文章首页缓存 5 分钟

    4 、文章内页缓存 1 天

    TP 实现方法：

    在当前模块配置中开启缓存静态，并在规则里 HTML_CACHE_RULES 加上

    'News:Index' => ['/{$_SERVER.REQUEST_URI|md5}', 100, null, true],

    'News:Show' => ['/{$_SERVER.REQUEST_URI|md5}', 86400, null, true],

    'Article:Index' => ['/{$_SERVER.REQUEST_URI|md5}', 300, null, true],

    'Article:Show' => ['/{$_SERVER.REQUEST_URI|md5}', 86400, null, true],

    搞定，收工。

    请 Laraval 写出主要实现方法。
